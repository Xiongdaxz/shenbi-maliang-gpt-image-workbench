Shenbi Maliang GPT Image Workbench — это рабочая среда для генерации и редактирования изображений ИИ, рассчитанная на частные командные развертывания. Поддерживаются подписочные аккаунты ChatGPT, OpenAI-совместимые API, CPA-прокси и частные endpoints для изображений.

## ✨ Кратко

- 🖼️ Генерация изображений в диалоге и продолжение редактирования существующих изображений.
- 🧭 Разделение пользовательского рабочего пространства и админ-консоли.
- 🧩 Единое управление каналами изображений, аккаунтами команды, ассетами, кейсами и шаблонами промптов.
- 🔐 Данные выполнения по умолчанию хранятся локально, что удобно для частного развертывания.

## 🚀 Быстрый старт

```bash
bun install
bun run build
bun run start
```

После запуска откройте `http://127.0.0.1:8787`. Админ-консоль доступна по адресу `http://127.0.0.1:8787/config`.
