Shenbi Maliang GPT Image Workbench محیطی برای تولید و ویرایش تصویر با هوش مصنوعی است که برای استقرار خصوصی در تیم‌ها طراحی شده است. این پروژه از حساب‌های اشتراک ChatGPT، APIهای سازگار با OpenAI، پراکسی‌های CPA و endpointهای خصوصی تصویر پشتیبانی می‌کند.

## ✨ نمای کلی

- 🖼️ تولید تصویر به صورت گفت‌وگویی و ادامه ویرایش بر پایه تصاویر موجود.
- 🧭 جداسازی محیط کاری کاربران از کنسول مدیریت.
- 🧩 مدیریت یکپارچه کانال‌های تصویر، حساب‌های تیم، دارایی‌ها، نمونه‌ها و قالب‌های پرامپت.
- 🔐 داده‌های اجرا به طور پیش‌فرض محلی ذخیره می‌شوند و برای استقرار خصوصی مناسب‌اند.

## 🚀 شروع سریع

```bash
bun install
bun run build
bun run start
```

پس از اجرا، `http://127.0.0.1:8787` را باز کنید. کنسول مدیریت در `http://127.0.0.1:8787/config` قرار دارد.
