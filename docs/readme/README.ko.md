Shenbi Maliang GPT Image Workbench는 팀 내부 프라이빗 배포를 위해 설계된 AI 이미지 생성 및 이미지 편집 워크벤치입니다. ChatGPT 구독 계정, OpenAI 호환 API, CPA 프록시, 프라이빗 이미지 엔드포인트 등을 연결할 수 있습니다.

## ✨ 한눈에 보기

- 🖼️ 대화형 이미지 생성과 기존 이미지 기반 편집을 지원합니다.
- 🧭 사용자 작업 공간과 관리자 콘솔을 분리했습니다.
- 🧩 이미지 채널, 팀 계정, 에셋, 사례, 프롬프트 템플릿을 통합 관리합니다.
- 🔐 런타임 데이터는 기본적으로 로컬에 저장되어 프라이빗 배포에 적합합니다.

## 🚀 빠른 시작

```bash
bun install
bun run build
bun run start
```

실행 후 `http://127.0.0.1:8787`을 열어 사용합니다. 관리자 콘솔은 `http://127.0.0.1:8787/config`입니다.
