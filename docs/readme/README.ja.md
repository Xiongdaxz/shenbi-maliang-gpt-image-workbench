Shenbi Maliang GPT Image Workbench は、チーム内でのプライベート運用を想定した AI 画像生成・画像編集ワークベンチです。ChatGPT サブスクリプションアカウント、OpenAI 互換 API、CPA プロキシ、プライベート画像エンドポイントなどを利用できます。

## ✨ 概要

- 🖼️ 会話形式で画像を生成し、既存画像の編集も継続できます。
- 🧭 ユーザー用ワークスペースと管理コンソールを分離しています。
- 🧩 画像チャネル、チームアカウント、素材、ケース、プロンプトテンプレートを一元管理できます。
- 🔐 ランタイムデータはローカル保存を基本とし、プライベート運用に向いています。

## 🚀 クイックスタート

```bash
bun install
bun run build
bun run start
```

起動後は `http://127.0.0.1:8787` を開きます。管理コンソールは `http://127.0.0.1:8787/config` です。
