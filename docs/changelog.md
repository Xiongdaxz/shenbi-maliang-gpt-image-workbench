# 更新日志

记录系统近期面向用户的功能更新、体验优化和问题修复。

## v0.1.54 - 2026-07-21

- 我的图片、素材库和灵感空间改为分批加载与虚拟滚动，大量内容下首屏打开、筛选和继续浏览更轻快。
- 聊天内选择素材或灵感不再预先加载全部内容，搜索、分类筛选和换一批推荐会按需请求，进入创作更快。
- 从全局搜索或图库进入图片编辑时，只加载当前图片和附近内容，长会话中切换图片更稳定。
- 图片预览的默认滚轮行为改为平移查看，放大后的大图浏览更贴近直觉，也可继续在偏好中切换为缩放。
- 分享页结果图下载和来源图预览规则更清晰；后台请求日志也能区分用户主动取消和真实失败。

## v0.1.53 - 2026-07-20

- 修复公网或反向代理部署下，创建和撤销会话分享链接可能被误判为请求来源无效的问题。
- 分享链接管理继续保留跨站请求拦截，正常同源访问和安全防护更稳定。

## v0.1.52 - 2026-07-20

- 新增会话分享链接，可将当前可见对话生成只读快照；访客无需登录即可查看，登录用户也能保留自己的工作台。
- 设置页新增分享链接管理，可复制、打开、回到原对话、撤销单条链接或一次撤销全部链接；删除原对话后相关链接会同步失效。
- 分享页优化只读浏览体验，默认定位到最新生成结果，并补齐一键滚动、图片标题说明、缩略图导航和结果图下载。
- 对话、图片编辑、灵感空间、素材库和“我的图片”的下载文件名统一缩短，并优先使用对应的对话标题、灵感标题或素材名称。
- 编辑个人资料新增头像裁剪和最近头像记录，可快速切换历史头像，保存异常时也会尽量保护原头像文件。
- 消息菜单会根据可用空间自动选择展开方向并保持右侧对齐；图片时间轴、帮助中心和更新日志搜索等细节继续优化。

## v0.1.51 - 2026-07-16

- 新增帮助中心，覆盖创作、编辑、素材、批量下载和设置等常用操作，并支持搜索、图示和多语言说明。
- 灵感空间新增贡献榜和更清晰的入口，方便发现活跃贡献者与优质共享内容。
- 图片生成和编辑过程中可直接取消当前任务，取消后会尽量恢复已提交内容，并减少未完成结果残留。
- 图片预览新增个人偏好，可设置默认适应窗口或原始尺寸打开，也可选择滚轮缩放或平移浏览。
- 设置页更新日志支持分页加载，发布包会随带更新日志，管理员也可从项目日志同步版本说明。
- 素材共享、审核状态和批量加入素材的文案更准确；删除对话和账号时增加二次确认保护。
- 侧边栏切换、生成中提示动画和多处弹窗布局继续优化，日常操作更顺滑稳定。

## v0.1.50 - 2026-07-14

- 图片生成和编辑任务在服务重启后会自动接管恢复，减少长任务中断后需要手动重来的情况。
- 任务恢复和手动重试增加接管保护，避免同一个任务被重复执行或重复返回结果。
- 新安装环境会自动带上常用灵感参考链接；用户删除后不会反复补回。
- 后台品牌设置新增 GitHub 入口开关，可按部署场景隐藏用户设置“关于”里的仓库入口。
- 图片预览中的复制提示词反馈文案更准确。

## v0.1.49 - 2026-07-13

- 侧边栏搜索升级为全局搜索，可同时查找聊天、图片、素材、灵感和提示词表单。
- 搜索结果新增分类数量和快捷切换，键盘方向键、回车和快捷键操作更顺手。
- 图片、素材和灵感搜索结果可直接打开对应预览；图片结果也可直接进入编辑工作台。
- 搜索历史更精准，只记录有结果的关键词，输入新关键词时不再干扰当前搜索。
- 提示词表单搜索展示表单图标和简介，跨库查找内容时更容易识别目标。

## v0.1.48 - 2026-07-12

- 新增多图 ZIP 下载，可选择原图、预览图或缩略图，并附带图片信息清单。
- 批量操作增加数量上限、结果汇总和异常提示，处理大量图片时更清楚、更可控。
- 图片编辑选择模式下可用滚轮调整画笔大小，局部编辑选择更顺手。
