# 更新日志

记录系统近期面向用户的功能更新、体验优化和问题修复。

## v0.1.38 - 2026-06-19

- 配置后台拆分为更清晰的模块结构，后续维护和扩展更稳定。
- 设置入口的页面组织更清楚，常用配置项后续调整会更顺手。

## v0.1.37 - 2026-06-18

- 设置页新增“关于”分区，可直接查看当前版本、开源仓库和许可证入口。
- 更新日志继续保留在设置页内展示，查看版本变化更集中。
- 最近聊天分组的收起展开动效更平滑，侧边栏浏览体验更顺手。
- 切换 AI 优化风格时会更稳定保留原始提示词的自动优化行为，减少来回调整。

## v0.1.36 - 2026-06-17

- 最近聊天支持置顶，并按“已置顶 / 最近”分组查看，常用对话更容易回到手边。
- 侧边栏收起、悬浮提示和更多菜单交互进一步打磨，桌面端浏览与操作更顺手。
- AI 优化在多图生成时会更明确保留“分别生成多张独立图片”的要求，减少被误写成拼图或拼贴的情况。
- 对话区自动滚动、弹窗层级和浅色模式细节继续优化，长对话和弹窗操作更稳定。

## v0.1.35 - 2026-06-16

- README 项目定位补充 ChatGPT `gpt-image-2` 生图平台说明，方便新用户快速理解项目用途。
- 输入框粘贴图片支持仅作为本次编辑引用使用，也可以按个人设置自动保存到素材库。
- 素材上传和加入素材库增加重复识别，重复素材会直接提示已有内容，减少素材库冗余。
- 图片编辑和对话输入区的素材选择器改为抽屉式动效，选择素材、关闭面板和清空输入更顺手。
- 聊天消息新增更多菜单，可查看消息时间；生成中和消息列表滚动定位更稳定。
- 优化侧边栏收起展开、移动端布局和 AI 优化风格选择器，长文案和小屏显示更自然。

## v0.1.34 - 2026-06-15

- 品牌 Logo 和浏览器图标加载优化，默认图标改为轻量资源，后台上传的品牌图也会生成缩略图和预览图。
- 图片生成和编辑完成后会后台预生成续改建议，打开图片时更快看到可继续编辑的方向。
- 图片接口返回异常时会展示更具体的错误信息，便于判断是上游拒绝、返回不完整还是后处理失败。
- 对话标题增加生成状态，后台生成标题时列表和详情页显示更稳定。
- AI 优化的自定义补充说明改为手动触发，仅点击自定义优化按钮时生效，避免影响普通风格优化。
- 优化空白页 Logo 动效和品牌资源缓存，减少重复加载和闪烁。

## v0.1.33 - 2026-06-15
